## ⚡ Quick Start

### 1️⃣ Clone

```bash id="r6m3qp"
git clone https://github.com/Vinothkumar-SV/WebScrapingUsing_AIAgent.git
cd WebScrapingUsing_AIAgent
```

### 2️⃣ Install

```bash id="q8j4tx"
npm install
```

### 3️⃣ Launch

```bash id="v2k9de"
npm run dev
```

### 4️⃣ Open

```text id="y1z7uh"
http://localhost:5173
```
